# 项目调试部署：


## 🚀 配置工程环境：

- `Node.js` 版本 18.14.2
- `@vue/cli` 版本 5.0.8
- `npm` 版本 9.5.0

进入项目工程目录，注意这是指有 `package.json` 的目录层级。在当前层级目录执行以下指令安装项目运行所需要的依赖包：

```
npm install
```

### 编译本地运行：

等待 `install` 完成之后，再执行以下命令启动项目。

```
npm run serve
```

### 打包部署：

运行以下指令，打包当前工程，发布到生产环境。

```
npm run build
```

### 修复：
```
npm run lint
```
